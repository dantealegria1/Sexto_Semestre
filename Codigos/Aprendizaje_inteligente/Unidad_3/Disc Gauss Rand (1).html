Modelo Discriminante Gausiano

Un poco de contexto

El Modelo Discriminante Gaussiano (GDA) es un algoritmo de aprendizaje supervisado utilizado para la clasificacin de datos. Se basa en la suposicin de que los datos de cada clase se distribuyen segn una distribucin gaussiana (tambin conocida como distribucin normal).

El Modelo Discriminante Gaussiano (GDA) es adecuado para problemas de clasificacin donde las variables de entrada son continuas y siguen una distribucin gaussiana. En este ejemplo, se utilizar el conjunto de datos de aleatorios.

# Funcin para calcular la probabilidad de clase (phi)
def calculate_phi(class_samples):
    return len(class_samples) / total_samples

# Funcin para calcular la media (mu) de una clase
def calculate_mu(class_samples):
    return np.mean(class_samples)

# Funcin para calcular la matriz de covarianza (sigma) de una clase
def calculate_sigma(class_samples):
    return np.cov(class_samples, rowvar=False)

# Funcin de densidad de probabilidad gaussiana unidimensional
def gaussian_pdf(x, mu, sigma):
    return (1 / (np.sqrt(2 * np.pi) * sigma)) * np.exp(-(x - mu)**2 / (2 * sigma**2))
  1. Clculo de la probabilidad de clase (phi):
  • El cdigo comienza calculando la probabilidad de cada clase en el conjunto de datos de entrenamiento. Esto se hace contando cuntas veces aparece cada clase en el conjunto de datos y dividiendo ese nmero por el total de ejemplos de entrenamiento (m).
  • En la implementacin, phi es un vector que contiene las probabilidades de cada clase.
  1. Clculo de las medias y matrices de covarianza para cada clase (mu y sigma):
  • Luego, se calculan las medias y matrices de covarianza para cada clase. Estos parmetros se utilizan para modelar la distribucin de datos de cada clase como una distribucin gaussiana multivariante.
  • Mu es una matriz donde cada fila representa la media de las caractersticas para una clase especfica.
  • Sigma es una matriz tridimensional donde cada matriz de dos dimensiones (input_feature x input_feature) representa la matriz de covarianza para una clase especfica.

Creamos la funcin para predecir si un punto pertenece a una clase u a otra

In[30]:
# Funcin para predecir la clase de un punto de prueba
def predict_class(x_test, phi1, mu1, sigma1, phi2, mu2, sigma2):
    # Calcular la probabilidad de pertenencia a cada clase
    prob_class1 = phi1 * gaussian_pdf(x_test, mu1, sigma1)
    prob_class2 = phi2 * gaussian_pdf(x_test, mu2, sigma2)
    
    # Devolver la clase con la probabilidad ms alta
    return 1 if prob_class1 > prob_class2 else 2

En este caso, se calcula la probabilidad de que el punto de prueba "x_test" pertenezca a una clase y a la otra, y se devuelve la clase de la cual, la probabilidad sea mayor, ms adelante se tiene una grfica que muestra una linea de decisin que muy aproximadamente representa las regiones que le pertenecen a una clase y a la otra.

En el caso de un problema de clasificacin binaria como el que estamos tratando aqu, donde estamos modelando dos clases utilizando distribuciones gaussianas, no necesariamente existe una lnea clara que divida perfectamente las dos clases en el espacio de caractersticas, aunque en este caso estamos tratando de mostrar una que es muy aproximada.

En el caso de las distribuciones gaussianas, la "lnea" que divide las dos clases se puede conceptualizar como el lmite de decisin, que es la regin en la que las probabilidades de pertenecer a una clase u otra son iguales. En un problema ideal, este lmite de decisin sera una lnea recta en el caso de una dimensin, o un hiperplano en espacios de caractersticas de dimensiones superiores. Sin embargo, en la prctica, esta lnea puede ser ms compleja y no necesariamente lineal.

En nuestro caso, con dos distribuciones gaussianas, el lmite de decisin se puede visualizar como el punto medio entre las medias de las dos distribuciones. Sin embargo, debido a la superposicin de las distribuciones y la naturaleza de las distribuciones gaussianas, este lmite puede no ser tan claro y puede haber cierta ambigedad en la clasificacin de puntos que se encuentran cerca de esta regin de superposicin.
