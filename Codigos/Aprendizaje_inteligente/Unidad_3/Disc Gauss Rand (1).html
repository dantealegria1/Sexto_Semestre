Modelo Discriminante Gausiano¶

Un poco de contexto¶

El Modelo Discriminante Gaussiano (GDA) es un algoritmo de aprendizaje supervisado utilizado para la clasificación de datos. Se basa en la suposición de que los datos de cada clase se distribuyen según una distribución gaussiana (también conocida como distribución normal).

El Modelo Discriminante Gaussiano (GDA) es adecuado para problemas de clasificación donde las variables de entrada son continuas y siguen una distribución gaussiana. En este ejemplo, se utilizará el conjunto de datos de aleatorios.

# Función para calcular la probabilidad de clase (phi)
def calculate_phi(class_samples):
    return len(class_samples) / total_samples

# Función para calcular la media (mu) de una clase
def calculate_mu(class_samples):
    return np.mean(class_samples)

# Función para calcular la matriz de covarianza (sigma) de una clase
def calculate_sigma(class_samples):
    return np.cov(class_samples, rowvar=False)

# Función de densidad de probabilidad gaussiana unidimensional
def gaussian_pdf(x, mu, sigma):
    return (1 / (np.sqrt(2 * np.pi) * sigma)) * np.exp(-(x - mu)**2 / (2 * sigma**2))
  1. Cálculo de la probabilidad de clase (phi):
  • El código comienza calculando la probabilidad de cada clase en el conjunto de datos de entrenamiento. Esto se hace contando cuántas veces aparece cada clase en el conjunto de datos y dividiendo ese número por el total de ejemplos de entrenamiento (m).
  • En la implementación, phi es un vector que contiene las probabilidades de cada clase.
  1. Cálculo de las medias y matrices de covarianza para cada clase (mu y sigma):
  • Luego, se calculan las medias y matrices de covarianza para cada clase. Estos parámetros se utilizan para modelar la distribución de datos de cada clase como una distribución gaussiana multivariante.
  • Mu es una matriz donde cada fila representa la media de las características para una clase específica.
  • Sigma es una matriz tridimensional donde cada matriz de dos dimensiones (input_feature x input_feature) representa la matriz de covarianza para una clase específica.

Creamos la función para predecir si un punto pertenece a una clase u a otra¶

# Función para predecir la clase de un punto de prueba
def predict_class(x_test, phi1, mu1, sigma1, phi2, mu2, sigma2):
    # Calcular la probabilidad de pertenencia a cada clase
    prob_class1 = phi1 * gaussian_pdf(x_test, mu1, sigma1)
    prob_class2 = phi2 * gaussian_pdf(x_test, mu2, sigma2)
    
    # Devolver la clase con la probabilidad más alta
    return 1 if prob_class1 > prob_class2 else 2

En este caso, se calcula la probabilidad de que el punto de prueba "x_test" pertenezca a una clase y a la otra, y se devuelve la clase de la cual, la probabilidad sea mayor, más adelante se tiene una gráfica que muestra una linea de decisión que muy aproximadamente representa las regiones que le pertenecen a una clase y a la otra.

En el caso de un problema de clasificación binaria como el que estamos tratando aquí, donde estamos modelando dos clases utilizando distribuciones gaussianas, no necesariamente existe una línea clara que divida perfectamente las dos clases en el espacio de características, aunque en este caso estamos tratando de mostrar una que es muy aproximada.

En el caso de las distribuciones gaussianas, la "línea" que divide las dos clases se puede conceptualizar como el límite de decisión, que es la región en la que las probabilidades de pertenecer a una clase u otra son iguales. En un problema ideal, este límite de decisión sería una línea recta en el caso de una dimensión, o un hiperplano en espacios de características de dimensiones superiores. Sin embargo, en la práctica, esta línea puede ser más compleja y no necesariamente lineal.

En nuestro caso, con dos distribuciones gaussianas, el límite de decisión se puede visualizar como el punto medio entre las medias de las dos distribuciones. Sin embargo, debido a la superposición de las distribuciones y la naturaleza de las distribuciones gaussianas, este límite puede no ser tan claro y puede haber cierta ambigüedad en la clasificación de puntos que se encuentran cerca de esta región de superposición.
